# Advent of Code 2018 (TypeScript)

Test-first Advent of Code workspace using TypeScript and Vitest.

## Included structure

- src/dayx/dayx.ts: starter solution file
- src/dayx/dayx.test.ts: starter tests
- src/dayx/input.txt: starter input file
- src/utility: shared helpers and extension methods used across days

## Commands

- pnpm i
- pnpm test

## Notes

- Add new days under src/dayN with dayN.ts, dayN.test.ts, and input.txt.
- Tests are configured to run from src using Vitest.
